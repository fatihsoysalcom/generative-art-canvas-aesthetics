# Generative Art Canvas Aesthetics

This example demonstrates how simple JavaScript code can be used to create generative art on an HTML canvas. It illustrates the article's concept of programming as an artistic medium by generating unique, abstract visual patterns through programmatic rules and aesthetic parameter choices, showcasing creativity in code. Each run produces a new composition of colored, translucent circles, highlighting the beauty and expressiveness achievable with code.

## Language

`html`

## How to Run

Save the code as `index.html`. Open `index.html` in any modern web browser (e.g., Chrome, Firefox, Edge).
